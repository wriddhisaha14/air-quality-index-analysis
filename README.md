# Air Quality Index (AQI) Analysis

## Overview
This project presents a comprehensive analysis of air quality patterns across various regions using AQI data. It covers pollutant trends, time-based fluctuations, and spatial comparisons, compiled as part of a college-level data analytics submission.

## Objective
To analyze air quality metrics through visualizations, descriptive statistics, and data interpretation—highlighting pollution trends, severity levels, and regional effects based on the Air Quality Index (AQI).

## Data Source
- *Origin* : Kaggle
- *Format* : Raw CSV 
- *Key Metrics* :
  - AQI Score
  - PM2.5, PM10
  - CO, SO₂, NO₂, O₃ concentrations
  - Time and Location attributes

## Tools & Technologies
All analysis, code, plots, and outputs were created and compiled in a Jupyter Notebook, later exported as a formatted PDF document.
- *Python (Pandas, Matplotlib, Seaborn)*
- *Jupyter Notebook*
- *PDF Export for Presentation*

## Deliverable
This repository contains the complete analytical report in a single document. The PDF includes:
- Data Cleaning Steps
- Code and Explanations
- Statistical Summaries
- Visualizations
- Insights and Conclusions
